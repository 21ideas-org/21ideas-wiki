# Введение в вики 21ideas

Эта вики — синтезированная база знаний по Биткоину на русском языке. Все страницы написаны на основе оригинальных русскоязычных источников сайта 21ideas.org.

---

## Что это такое

21ideas.org — некоммерческий образовательный проект, посвящённый только Биткоину. За годы работы на сайте накопились сотни материалов: переводы книг, серии статей, руководства, история. Эта вики — структурированный дистиллят этих знаний.

**Вики не заменяет оригинальные материалы** — она служит картой и точкой входа. Для глубокого погружения читайте первоисточники по ссылкам на каждой странице.

## О качестве

Каждая страница содержит метаданные в YAML frontmatter:

- `quality: synthesized` — написано на основе нескольких источников
- `quality: reference` — описание конкретного источника (книги, серии)
- `completeness: high/medium/low` — степень охвата темы

Страницы с `completeness: medium` можно расширить при появлении новых источников.

## Как обновлять вики

1. Добавьте новый источник в `raw/`
2. Попросите LLM-ассистента обработать его
3. Ассистент обновит существующие страницы и при необходимости создаст новые
4. Обновите `index.md` при добавлении страниц
5. Добавьте запись в `log.md`

Технические подробности – в файлах README и WIKI-GUIDE
