21wiki открыта для участия. Есть три способа сделать вклад — выберите тот, что подходит по времени и техническим навыкам.

## Вариант 1 — Предложить (без кода)

Самый простой вклад. Откройте GitHub Issue, чтобы:

- **Предложить недостающую тему** — что-то, чего пока нет в вики
- **Сообщить об ошибке** — неточный факт, сломанная ссылка или устаревшие сведения
- **Запросить языковой паритет** — тема есть на английском, но не на русском (или наоборот)

👉 [Открыть issue](https://github.com/21ideas-org/21ideas-wiki/issues/new/choose)

## Варинат 2 — Добавить исходный файл

Вики строится на неизменяемых исходных файлах в `raw/` — все они с [21ideas.org](https://21ideas.org). Если у вас есть на примере качесвтенный материал, который будет полезен вики, вы можете добавить его в исходную библиотеку.

Правила приёма источников:
- Без явного одобрения мейнтейнера принимаются только материалы **21ideas.org**
- Первичные биткоин-источники (BIP, фундаментальные материалы, тексты Сатоши) рассматриваются в индивидуальном порядке — сначала [откройте issue](https://github.com/21ideas-org/21ideas-wiki/issues/new/choose)
- Сторонние блоги, контент бирж и материалы об альткоинах **не принимаются**

После добавления исходного файла мейнтейнер займётся синтезом вики-страниц — или вы можете перейти к Варианту 3.

## Вариант 3 — Генерация страниц с помощью ИИ

Полный путь контрибьютора: форкните репозиторий, запустите ИИ-агент локально (Claude Code, Cursor или Gemini CLI) на полном содержимом репозитория, сгенерируйте страницы для обоих языковых слоёв, проверьте линтером и откройте PR.

Этот трек требует локальной среды. Рабочий процесс, правила и промпт для агента — на английском в гайде на GitHub:

👉 [docs/CONTRIBUTING.md](https://github.com/21ideas-org/21ideas-wiki/blob/main/docs/CONTRIBUTING.md)
