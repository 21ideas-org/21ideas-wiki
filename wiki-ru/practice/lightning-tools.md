# Lightning Network: инструменты и кошельки

[[ru/concepts/lightning-network|Lightning Network]] — сеть платёжных каналов поверх Биткоина. Мгновенные транзакции с минимальными комиссиями. Для работы с Lightning нужен отдельный кошелёк.

---

## Мобильные кошельки

### Phoenix Wallet

**Лучший выбор для большинства пользователей.**

- iOS и Android
- Некастодиальный: вы контролируете ключи
- Автоматическое управление каналами (не нужно понимать, как работают каналы)
- Поддержка BOLT 12 (обновлённый стандарт Lightning-инвойсов)
- Разработчик: ACINQ (Франция)

Phoenix — наиболее сбалансированный вариант: некастодиальный, но при этом простой в использовании.

---

## Инструменты для продвинутых пользователей

### LNbits

«Швейцарский нож» Lightning-экосистемы.

- Набор приложений для вашего Lightning-узла
- Плагины: точка продаж (PoS), сплит-платежи, подписки, фаустет (кран для тестирования), Discord/Telegram-бот
- Можно развернуть на собственном сервере или использовать публичный инстанс
- Подходит для Lightning-экосистемы вашего сообщества или бизнеса

### Alby

Браузерное расширение для Lightning.

- Интеграция с веб-сайтами: оплата одним кликом
- Поддержка Nostr: автоматические zap-платежи
- Lightning-адрес в формате `имя@getalby.com`
- Работает с любым Lightning-узлом

---

## Дополнительные инструменты

### ZapPlanner

Сервис для настройки регулярных Lightning-платежей (подписок). Поддерживает расписание автоматических транзакций без необходимости держать кошелёк открытым.

### ZapGram

Telegram-бот для приёма и отправки биткоинов через Lightning прямо в чате. Подходит для сообществ и групп.

---

## Управление ликвидностью

Для продвинутых пользователей, запускающих собственный Lightning-узел:

- **Входящая ликвидность**: без неё нельзя принимать платежи
- **Балансировка каналов**: поддержание ликвидности на обеих сторонах
- **Loop и Pool**: сервисы Lighting Labs для управления ликвидностью

---

## С чего начать

1. Установите **Phoenix** на телефон
2. Получите первый биткоин через Lightning (с биржи или от друга)
3. Попробуйте сделать первый платёж в магазине, поддерживающем Lightning
