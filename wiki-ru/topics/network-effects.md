## Введение

Сетевой эффект — явление, при котором ценность продукта или сети растёт по мере увеличения числа её пользователей. [[ru/concepts/bitcoin|Биткоин]] обладает несколькими взаимоусиливающими сетевыми эффектами, которые делают его позицию всё более устойчивой.

Основа: [«Семь сетевых эффектов биткоина»](https://21ideas.org/sem-setevyh-effektov-bitkoina).

## Семь сетевых эффектов (по Трейсу Мейеру)

### 1. Спекуляция

Биткоин — новый класс активов с потенциалом роста и высокой волатильностью. Это привлекает спекулянтов с высокой толерантностью к риску. Спекулятивный спрос обеспечивает ликвидность и цену, которая привлекает всех остальных участников.

### 2. Принятие бизнесами

Бизнесы принимают Биткоин, избегая [[ru/glossary#Комиссия (Transaction Fee)|комиссий]] платёжных карт (1–3%) и чарджбэков. По мере роста принятия — больше мест для трат → больше пользователей → больше бизнесов.

### 3. Принятие потребителями

Потребители получают возможность совершать платежи, недоступные другими способами (международные переводы без посредников, [[ru/concepts/privacy|приватность]]). Каждый новый пользователь ценнее предыдущих.

### 4. Безопасность

Цена → [[ru/concepts/mining|майнинг]] → [[ru/glossary#Хэшрейт|хешрейт]] → безопасность. Чем выше цена, тем больше майнеров, тем дороже [[ru/glossary#Атака 51%|атака 51%]]. Безопасность → доверие → цена. Положительная петля обратной связи.

### 5. Предпочтение разработчиков

Биткоин — простые и предсказуемые правила, [[ru/concepts/bitcoin-core|открытый код]], [[ru/concepts/decentralization|децентрализованная]] инновация без разрешений. Наибольший пул талантов, наибольшее количество ревьюеров кода, наибольшая безопасность. Альткоины не могут конкурировать за разработчиков — Биткоин предлагает крупнейшее сообщество и наиболее зрелую инфраструктуру.

### 6. Финансирование

Биткоин постепенно поглощает рынок традиционных банковских сервисов: денежные переводы, микроплатежи, P2P-кредитование. Каждый доллар финансирования в биткоин-инфраструктуру укрепляет сетевой эффект.

### 7. Мировая резервная валюта

Логическое завершение всех предыдущих эффектов. Если [[ru/concepts/scarcity|предложение ограничено]], безопасность высока, принятие растёт — Биткоин становится базовым активом для урегулирования всех других [[ru/glossary#Транзакция|транзакций]].

## Эффект Линди

Эффект Линди: чем дольше нечто существует, тем дольше оно будет существовать. Биткоин работает с 2009 года без перерыва. Каждый год без критического сбоя повышает вероятность существования в следующие годы.

Альткоины не могут конкурировать с этим активом: они слишком молоды и не имели возможности доказать надёжность через годы атак, [[ru/concepts/forks|форков]] и рыночных циклов.

## Самоусиливающаяся петля

```text
Больше пользователей
    → выше ликвидность
    → больше принятие
    → выше безопасность
    → выше цена
    → больше разработчиков
    → лучше продукт
    → больше пользователей
```

Это фундаментальная причина, по которой Биткоин сохраняет доминирование несмотря на тысячи конкурентов.
